## How to install and run locally ?

```
$ git clone https://github.com/deshwaninikil/todo_app.git
$ cd  todoapp
$ npm install
$ npm start

```

## Features Added

- Login Page
- SignUp Page
- Add Todo page
- Todo List Page
- Error Page
- Toast and Loaders
- Responsive

## Built with -

- NextJS
- NextJS Backend
- Next Auth
- Vanilla CSS
- Prisma
